Mini-project: Trailgage - Smart car

TrailGage is a smart luggage carrier based on a smart car platform that allows the user to navigate the smart car with their device. This allows the user to move around freely without having to carry/drag their luggage.

The smart car uses a WiFi system to allow the user to connect. Once a user has connected to a TrailGage, they will be able to control the carrier with their phone using the TrailGage app. An obstacle avoidance mechanism is integrated to avoid collision with obstacles which are in front of the smart car.

GitHub : TrailGage
Technologies used : Arduino, Android, C++








Mini-project: EventNode - Event planner

EventNode is a platform where people can participate in events created by other users or create an event to allow users to join. This will allow people to find friends and take initiative to create something exciting and fun. The attendees are able to rate the event and comment on the event before and after an event has occurred.

Technologies used : Vue, Bootstrap, Express, Node



Mini-project: Dentistimo - Dental clinic system

The Dentistimo is a distributed system which allows Dental clinics to post available time slots so that the users can reserve a time slot for their appointment. If a dental clinic has free time slots, it will be shown in the Dentistimo map to show the users clinics by availability.

Technologies used : MQTT, Vue, MongoDB



Project: Steering wheel angle computer

This project calculates the Steering Wheel angle using an algorithm to avoid blue and yellow cones which are in front of the car. It uses object detection and tracking to identify the cones in order to compute the steering wheel angle.

Technologies used : OpenCV, CMake, Docker
